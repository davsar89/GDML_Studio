# Sample GDML Data

Three GDML files are available in the project root directory:

1. `../BgoDetModel_v2_00.gdml`                  (158K) - BGO detector model
2. `../fermi_simple_elements_satellite.gdml`     (7.7K) - Fermi satellite simple geometry
3. `../NaiDetModelWithMLI_v2_00.gdml`            (167K) - NaI detector model with MLI

These files follow the GDML schema and contain <define>, <materials>,
<solids>, and <structure> sections for testing the parser.
